# Note: this program is simply intended to demonstrate different techniques employed when building an app in Android Studio #

Generates lottery numbers based on a statistical analysis of previous winning lottery numbers.

Implements Model-View-Controller structure.

External libraries include Jsoup, Firebase, & MP Android Chart.

Autonomously checks/updates Firebase in background.

Portrait/Landscape views locked for utilization.
